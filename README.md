# Few-Shot RVOS Datasets

This repository provides dataset splits, referring expressions, segmentation annotations, and evaluation metadata for Few-Shot Referring Video Object Segmentation (FS-RVOS).

The datasets support category-disjoint few-shot evaluation, direct cross-domain evaluation, and multi-object referring video segmentation. They are constructed and used in our paper:

> **RSCM: Reliable Support Construction Model for Few-Shot Referring Video Object Segmentation**

The repository includes Mini-Ref-YouTube-VOS, Mini-Ref-SAIL-VOS, Mini-DAVIS, and Mini-MeViS. It provides derived splits and annotations only; users should obtain original videos and source annotations from the corresponding official datasets.
